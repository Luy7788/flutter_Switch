使用方法
```

    width: 120, //宽度
    height: 44, //高度
    controller: mPageController, //下面的PageView的controller
    tabs: ['推荐', '热门'], 
    selectColor: Colors.white,
    unSelectColor: Colors.grey[300],
  ),
```
